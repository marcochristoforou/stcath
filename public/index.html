Thursday 16 January 2020

Heating and Hot Water System Replacement Nearly Done

This is lounge radiator. An old column radiator that previously had a massive brass pipe that went under the floor and back to the boiler. We've had to drop both pipes down from the circuit above, previously it had one pipe. It also feeds the hall radiator.

New pipework under bedroom 4. Outer pipes are the flow and return for the heating. Nearest pipe is the mains feed to the cylinder that will create the pressure we need.

All the pipework in the airing cupboard that controls the flow for turning heating or hot water or both on. You can have both on at the same time; unlike a combi. The system is pressurised like a combi which makes it more efficient. Electrics yet to go in. We'll be using Hive.

Boiler pipework. You can see the mains coming in bottom right. This will be the new main to replace the lead main. Still needs condenser waste to go in and lagging.

Electrics being done by Lee. Glad to see someone reading the instructions. This will allow Hive to control the values the activate the heating or hot water.
